# LogApp — Microservices Authentication App

## Tech Stack
- **Backend:** Spring Boot 4, Spring Security, JWT, OAuth2 (Google + GitHub)
- **Frontend:** React 18
- **Database:** PostgreSQL
- **Service Discovery:** Eureka Server
- **API Gateway:** Spring Cloud Gateway
